About

Access NFT is a low supply collection of 200 NFTS. Minting an Access NFT gives you access to the special tools we are building, more details on this in the FAQ section and in our discord server.

Frequently Asked Questions

  • What is Access NFT

    Access NFT is a collection of 200 NFTs. Having an Access NFT allows you to access our special tools.

  • What do we offer?

    We are building a platform that has tools to make your NFT experience a very good experience. The first tools we will be building is the Whitelist tool which allows you to get "real" discord invites to any NFT project.

  • What makes us unique?

    The Whitelist tool we are building will allow you get real discord invites. We have noticed that servers marks bot discord account as fake, so we have conquered this because we are using real discord accounts, and we also have a feature where each discord account can be verified in the discord server by automatically reacting to the verification message in the discord server

  • How can you access our tool?

    You can access this tool when you have an Access NFT

  • What blockchain will we be minting on?

    We will be minting on the solana blockchain.

  • Why Only 200 NFTs?

    We made the supply only 200 so that we can have the capacity to supply our services for every single person that mints.

    To get more details about our mint date and price, join out discord server
